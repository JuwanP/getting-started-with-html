The Basics of the Web

The web is made up of a collection of HTML documents. HTML stands for HyperText Markup Language. Here is a link to the w3schools page on HTML

When we go on our computers to browsers like, Chrome, FireFox, or Internet Explorer, and go to websites like www.google.com we send data to the Cloud which sends an HTTP requests to servers. The servers finds the appropriate files and sends the needed files back to our web browsers. The web browser then interprets and displays the files on our computers for the user to see.

The Main Document on the Web

HTML is the main type of document on the web. However, it's not the only type file, tht can exist on th web. Other files include: PDFs, Images, videos etc.

Major Concepts of the Web

The major concepts of the web are: The User, The Computer, The Internet, and Servers

HTML

An HTML is made up of text Content: what you see, Markup: what it looks like, and References to other documents: e.g. images and videos as well as links to other pages

Inline vs. Blocks

Inline elements such as: bold, emphasis, image, span are displayed with no line breaks. On the other hand, block elements such as: div and paragraph are displayed with an "invisible box" that separates them from surrounding elements, ands displays each one block element on a separate line.
